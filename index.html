Redes de Computadores: O Mundo Conectado

Esta seção inicial apresenta a base das Redes de Computadores, explicando o que elas são e por que são essenciais no mundo digital. Vamos entender como equipamentos e softwares colaboram para permitir o compartilhamento de recursos e a comunicação global, formando a espinha dorsal da internet.

O que é uma Rede de Computadores?

É um conjunto de equipamentos e softwares que se ligam e trabalham juntos.

O principal objetivo é permitir o compartilhamento de recursos (como impressoras) e a comunicação entre eles.

Tipos de Rede

As redes de computadores variam em tamanho e alcance, desde as mais pessoais até as que conectam continentes inteiros. Compreender esses tipos nos ajuda a classificar e entender as diferentes escalas de conexão digital.

Redes Pessoais (PAN)

Conectam dispositivos em curta distância, geralmente usando Bluetooth.

  • Ex: Celular e fone de ouvido sem fio.

Redes Locais (LAN)

Muito utilizadas em locais pequenos como casas e escritórios.

  • Geralmente de propriedade privada.

Redes Metropolitanas (MAN)

Conectam dispositivos em distâncias maiores, na mesma cidade ou bairro, interligando LANs.

Redes de Longa Distância (WAN)

Conectam dispositivos em diferentes locais no mundo, englobando várias LANs.

  • Ex: A Internet.

Desenhos das Redes: Topologias

A topologia de rede descreve o desenho físico de como os dispositivos estão conectados. Esse arranjo influencia diretamente a performance, a confiabilidade e a facilidade de gerenciamento da rede.

Barramento

Cada dispositivo conectado a um único cabo principal.

  • Simples, mas limitado; falhas no cabo afetam toda a rede.

Anel

Dispositivos conectados lado a lado, formando um círculo.

Estrela

Cada dispositivo se liga a um Hub ou Comutador central.

  • Mais comum, fácil de gerenciar; falha de um dispositivo não afeta outros.

Malha

Todos os dispositivos se conectam entre si.

  • Pode ser física (cabos) ou lógica; super resistente a falhas.

Os "Ajudantes" da Rede: Dispositivos

A infraestrutura de rede é composta por diversos dispositivos que trabalham juntos para garantir a transmissão eficiente e segura dos dados. Cada um possui uma função específica, essencial para o funcionamento da comunicação.

Repetidor

Dispositivo de duas portas que regenera sinais de rede para estender o alcance.

Hub

Atua como um repetidor multiporta, enviando cópias de cada pacote para todos os dispositivos conectados.

Ponte (Bridge)

Divide a rede em segmentos e pode filtrar/encaminhar pacotes de dados entre eles usando o endereço MAC.

Comutador (Switch)

Combina funcionalidade de ponte e hub, direcionando pacotes de dados diretamente ao dispositivo correto. Pode ser gerenciado ou não.

Roteador

Realiza a vinculação de redes de diferentes alcances, interpretando e filtrando pacotes de dados para encaminhá-los ao destino. Utiliza o IP do dispositivo.

A "Língua" da Rede: Padrões e Protocolos

Para que os dispositivos de rede possam se comunicar de forma padronizada e segura, eles seguem regras e acordos. Padrões governam o hardware e software, enquanto protocolos definem as condições e regras da comunicação.

Padrões de Rede

Fornecem uma estrutura que permite a interoperabilidade entre dispositivos.

  • **Ethernet (IEEE 802.3):** Padrão do setor para redes LAN, MAN e WAN, define estrutura de transmissão.
  • Existem versões como Fast Ethernet (100 Mbps), Gigabit Ethernet (1 Gbps), 10 Gigabit Ethernet (10 Gbps) e Terabit Ethernet (200-400 Gbps).

Protocolos de Rede

Conjunto de condições e regras que especificam como os dispositivos se comunicam.

  • **Endereço MAC:** Identifica o adaptador de rede no nível do hardware.
  • **Endereço IP:** Identifica o adaptador de rede no nível do software.
  • **Pacote de Dados (Datagrama):** Unidade de mensagem trocada, contendo dados e cabeçalhos.
  • **Roteamento:** Mecanismo que verifica pacotes de dados para o caminho de entrega correto.

Protocolos Essenciais de Comunicação

A internet funciona através de diversos protocolos que coordenam as diferentes formas de comunicação. Estes são alguns dos mais fundamentais, divididos em categorias para melhor compreensão.

Protocolos de Rede e Transporte

  • **TCP (Transmission Control Protocol):** Agrupa dados em pacotes, garante entrega segura e na ordem.
  • **IP (Internet Protocol):** Responsável pelo endereçamento do pacote de dados.
  • **UDP (User Datagram Protocol):** Protocolo sem conexão, baixa latência e tolerante a perda.

Protocolos de Aplicação e E-mail

  • **HTTP (Hypertext Transfer Protocol):** Usado para fornecer conteúdo de página da Web.
  • **FTP (File Transfer Protocol):** Usado para transferir arquivos entre computadores.
  • **POP3 (Post Office Protocol 3):** Protocolo de email para recebimento.
  • **SMTP (Simple Mail Transfer Protocol):** Usado para enviar emails.
  • **IMAP (Internet Message Access Protocol):** Permite gerenciar caixa de correio no servidor.

Protocolos de Segurança

  • **SSL (Secure Sockets Layer) / TLS (Transport Layer Security):** Fornecem conexão segura e criptografada.
  • **HTTPS:** Versão segura do HTTP usando TLS ou SSL.
  • **SSH (Secure Shell):** Protocolo criptográfico para conexão de dados segura em rede.
  • **Kerberos:** Protocolo de validação que fornece autenticação robusta.

Organizando a Conversa: Modelo TCP/IP

O Modelo TCP/IP é a base da arquitetura da internet, dividindo a comunicação em quatro camadas lógicas. Cada camada tem responsabilidades específicas, trabalhando em conjunto para garantir que os dados sejam transmitidos de forma eficiente e confiável.

  • Camada de Aplicativo: Lida com a comunicação do aplicativo ou processo (ex: HTTP, SMTP, IMAP).
  • Camada de Transporte: Responsável pela comunicação de host para host (TCP e UDP).
  • Camada de Internet: Responsável pela troca de datagramas, adiciona endereços IP de origem e destino (ex: IP, ICMP).
  • Camada de Acesso à Rede: Define como os dados são enviados pela rede (ex: ARP, MAC, Ethernet, DSL, ISDN).

Wi-Fi: A Magia Sem Fio

O Wi-Fi revolucionou a forma como nos conectamos à internet, permitindo mobilidade e flexibilidade. Baseado no padrão IEEE 802.11, ele permite que dispositivos se comuniquem sem a necessidade de cabos.

O que é Wi-Fi?

"Wireless Fidelity", significa que o dispositivo é compatível com o padrão **IEEE 802.11** para redes sem fio.

  • Permite conexão à internet sem cabos.

Frequências de Transmissão

  • **2.4 GHz:** Mais comum, maior alcance, mais sujeita a interferências.
  • **5 GHz:** Menos congestionada, menor interferência, melhor desempenho.

Evolução do Wi-Fi: Cada Vez Mais Rápido!

Desde 1997, os padrões Wi-Fi evoluíram, resultando em velocidades mais rápidas e maior cobertura.

  • **802.11b:** Padrão mais lento (11 Mbps).
  • **802.11g:** 2.4 GHz, mais rápido (54 Mbps).
  • **802.11n (Wi-Fi 4):** Aumenta velocidade e alcance (até 140 Mbps teóricos).
  • **802.11ac (Wi-Fi 5):** Opera em 5 GHz, velocidades de 433 Mbps até vários Gbps teóricos.
  • **802.11ax (Wi-Fi 6):** Padrão de última geração, taxas de dados e capacidade mais altas (até 9.6 Gbps), melhor em ambientes densos.
